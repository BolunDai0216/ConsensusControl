To execute the process please press the following keys:
"s" : square formation
"l" : leave the room
"c" : circle formation
"p" : go to the purple ball
"k" : push the purple ball to the purple square
"b" : leave the purple ball
"r" : go to the red ball
"t" : push the red ball to the red square
"o" : leave the red ball
"a" : enter room step 1
"e" : enter room step 2
"u" : enter room step 3
"d" : diamond formation
please wait at least 20 sec after covergence to initiate the next controller.
